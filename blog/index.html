1. Exportar Entradas del Blog

Después de hacer cambios en las entradas del blog, usa esta función para generar el código de las entradas actualizadas.

Instrucciones:
1. Haz clic en "Generar código de entradas"
2. Copia el código generado usando el botón "Copiar código"
3. En Canva, reemplaza solo la sección blog = [...]; en tu código original.

¡Código copiado con éxito!

2. Exportar HTML Completo

Genera el código HTML completo del blog con todas tus entradas actualizadas. Útil para guardar una copia de seguridad.

Instrucciones:
1. Haz clic en "Generar HTML completo"
2. Copia el código generado usando el botón "Copiar código"
3. Guarda el código en un archivo con extensión .html en tu computadora
4. Puedes abrir este archivo en cualquier navegador para ver tu blog

¡Código copiado con éxito!

3. Exportar Código Simplificado IA CANVA

Genera una versión resumida del código con los elementos esenciales. Útil para compartir con asistentes de IA como Canva Code.

Instrucciones:
1. Haz clic en "Generar código simplificado"
2. Copia el código generado usando el botón "Copiar código"
3. Comparte este código con un asistente de IA para que pueda entender la estructura y funcionalidad de tu blog

¡Código copiado con éxito!
